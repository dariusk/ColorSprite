Color Sprite
by Darius Kazemi
I was inspired by seeing Lokesh Dhakar's Color Thief, a JS library for grabbing palettes from images. Click on an image below to see its palette rendered as a sprite. Keep clicking to get different colors/configurations. (Original images CC licensed: sunset, duck, bar, mandala. The sprite that is generated is one of Oryx's awesome 2D 8x8 sprites. Should work in Chrome and Firefox.)
